# Vet Plus

 - Configurar o mongoDB (por padrão o nome esta vetplus)
 - Use o "nodemon app.js" para uma melhor experiencia (caso não tenha, { npm install -g nodemon } , para instalar globalmente)

**Server:** Node, Expresse e mongoDB
